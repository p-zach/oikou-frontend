# Oikou

A geography learning app.

"Oikou" comes from the Greek *oikoumene* or *ecumene*, originally meaning "the inhabited world" but today connoting the unified, modern world civilization.

Also see the [backend](https://github.com/p-zach/oikou-backend).

## Features

Current features:
- Learn capitals
- Spaced repetition: see difficult facts more frequently

Future features:
- Learn flags, neighbors, & location on a map
- More regions
- Review progress on an interactive map
- Practice multiple question types at once
